# SportsBiz Auction Management System

Enterprise-ready multi-tenant auction platform built with Laravel, Livewire, Alpine, Redis queues, and event broadcasting.

## Stack

- Laravel 12 (Laravel 10+ compatible architecture)
- Livewire (class-based components)
- Alpine.js for micro-interactions (countdown)
- MySQL schema design + indexed relations
- Broadcasting-ready events/channels (Reverb/Pusher compatible)
- Redis queue/broadcast defaults in environment
- Spatie Permission (role middleware + role seeding)
- Policies + tenant middleware for data isolation

## Roles

- SuperAdmin: platform owner
- Admin: tenant/client owner
- Team: bidder login users under an Admin tenant
- Public: read-only live auction viewer

## Implemented Modules

- SuperAdmin panel components
- Admin panel components for tournaments, players, teams, reports, and auction control
- Team panel components for auction room, squad, bid history
- Public live viewer component
- Core `AuctionEngine` with strict bid rules, anti-sniping, transaction-safe SOLD settlement
- Presence channels and broadcast events (`BidPlaced`, `PlayerSold`, `AuctionStarted`, `AuctionPaused`, `TimerExtended`)
- Bid endpoint with `FormRequest` validation and rate limiting
- Multi-tenant middleware + subscription middleware + policies
- Seeded demo data with roles and sample auction context

## Quick Start

1. Install dependencies:

```bash
composer install
```

2. Configure environment (`.env`):

- MySQL connection
- `QUEUE_CONNECTION=redis`
- `BROADCAST_CONNECTION=reverb` (or `pusher`)

3. Run setup:

```bash
php artisan migrate:fresh --seed
php artisan test --filter=AuctionEngineTest
```

4. Run app services:

```bash
php artisan serve
php artisan queue:work
```
